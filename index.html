Lenguajes de programación

Los lenguajes de programacíon son lenguajes con reglas definidas los cuales proporcionan al programador una manera de ontrolar el comportamiento físico o lógico de un sistema informático, de manera que se puedan obtener diversas clases de datos o ejecutar determinadas tareas.

Código binario

Cuando se crearon las primeras computadoras alimentadas de electricidad, estas eran muy lentas y poco potentes, por lo que su lenguaje de programación era simple; el código binario.

Este lenguaje solo se componía de 0 y 1, pero al ser demasiado confuso y propenso a ser mal escrito, se pensó en otro lenguaje distinto.
